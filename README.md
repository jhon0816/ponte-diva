# Ponte Diva Spa Landing

Landing page estática para Ponte Diva Spa, marca de uñas y belleza en Cali.

## Estructura

- `index.html`: página principal.
- `assets/img/logo.png`: logo original de la marca.
- `assets/img/logo-display.jpg`: versión optimizada usada en la página.
- `assets/img/favicon.png`: icono de pestaña generado desde el logo.
- `assets/img/work-*.jpeg`: fotos reales usadas en hero y galería.
- `assets/js/main.js`: menú móvil y año automático.

## Cómo verla

Abre `index.html` en el navegador. La página usa Tailwind CSS y Font Awesome por CDN, así que necesita conexión a internet para cargar estilos e iconos completos.
